\# 🚀 Network Intrusion Detection System (NIDS)



\## 📌 Overview

This project is a real-time Network Intrusion Detection System that uses Machine Learning models to detect cyber attacks from network traffic.



\---



\## 🧠 Features

\- Real-time packet capture using Npcap

\- Flow-based feature extraction

\- AI-based attack detection (LSTM, CNN, Autoencoder)

\- Live dashboard visualization

\- Attack classification (DDoS, Infiltration, etc.)



\---



\## 🏗️ Architecture



WiFi Network → Packet Capture → Feature Extraction → AI Models → Dashboard



\---



\## ⚙️ Tech Stack

\- Python

\- Flask

\- Scapy / PyShark

\- TensorFlow / Keras

\- HTML/CSS/JS (Dashboard)



\---



\## 🤖 AI Models Used

\- LSTM (Temporal analysis)

\- CNN (Pattern detection)

\- Autoencoder (Anomaly detection)



\---



\## 📊 Output

\- Real-time attack detection

\- Live graphs

\- Alert system
